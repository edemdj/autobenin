# Notes base de donnees AutoBenin

## Collections MongoDB
- **users** : locataires et proprietaires
- **cars** : vehicules verifies
- **bookings** : reservations + caution
- **payments** : transactions Mobile Money
- **reviews** : notes et commentaires

## Index recommandes
- `cars`: city, isAvailable, pricePerDay
- `bookings`: renter, owner, status
