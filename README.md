# AutoBenin — Location de voitures

## Demarrage rapide

### Backend
```bash
cd backend
npm install
cp .env.example .env
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

## Stack
- Frontend : React + Vite + Tailwind CSS
- Backend  : Node.js + Express + MongoDB
- Paiement : MTN MoMo / Moov Money
- Stockage : Cloudinary
- Auth     : JWT
